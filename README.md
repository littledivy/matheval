## `matheval`

Tiny and safe math expression evaluation for Deno.

### Features

- [x] Operators (+, -, *, /)
- [x] Integers
- [ ] Floats and Complex numbers
- [x] Variables
- [ ] Functions 
- [ ] Precedence

### Usage

```typescript
import evalutate from "https://deno.land/x/matheval/mod.ts";

evaluate("23 * 3") // 69
```
